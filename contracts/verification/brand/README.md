# TigerCoin Brand Assets

This folder contains official TigerCoin brand assets.

## Primary Logo
- SVG: `brand/logo/TigerCoin_Logo_v1.svg` (preferred for web/print)

## Avatar / Icon
- PNG: `brand/logo/TigerCoin_Logo_v1.png` (recommended for avatars)

## Usage Rules (Simple)
- Do not distort, stretch, or rotate the logo.
- Do not recolor or add effects that change the identity.
- Use the SVG whenever possible for crisp rendering.
