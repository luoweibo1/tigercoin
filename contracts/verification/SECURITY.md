# Security & Anti-Fraud Notice (TigerCoin / TIGER)

## Official Sources
The only official sources for TigerCoin (TIGER) are:
- Official website: https://tiger-coin.com
- This GitHub repository

If any information conflicts with the official website, treat it as **unofficial**.

## Never Trust DMs
Project representatives will **never** contact you first via private messages to:
- request funds,
- request private keys / seed phrases,
- request “signature approvals” unrelated to clearly described on-chain actions,
- ask you to “verify” your wallet by sending assets.

## Reporting
If you see impersonation or phishing, do not interact. Verify via the official website first.
